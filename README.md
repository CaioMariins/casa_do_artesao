# Dashboard Casa do Artesão

Projeto de visualização de dados desenvolvido com Streamlit.

## Estrutura do Dataset

Os formulários de artesãos fixos e visitantes foram unificados em um único dataset.

### Coluna de categorização

- tipo_cadastro
  - fixo
  - visitante

### Campos exclusivos de artesãos fixos

- raca
- inscricao_fazenda
- inscricao_cgm
- numero_barraca
- cnae
- metragem

## Valores padronizados

### genero

- mulher
- homem
- pessoa_trans
- outros

### estado_civil

- solteiro
- casado
- viuvo
- divorciado

### raca

- preto
- pardo
- branco
- amarelo
- indigena

### renda_artesanato

- 10%
- 30%
- 50%
- 70%
- 100%

### feira

- Campo São Bento
- Praça Getúlio Vargas
- Praça do Zumbi
- Praça César Tinoco
- Orla de São Francisco
- Largo do Marrão
- Outra

### Campos booleanos

Os seguintes campos devem utilizar apenas:

- true
- false

Campos:
- pcd
- aposentado

- outra_renda
